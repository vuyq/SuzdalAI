# SuzdalAI

Простой AI помощник, который ходит к GigaChat. Работает на основе RAG.

## Чтобы установить совой API-ключ. 

Пример простой конфигурации представлен ниже:

GIGACHAT_AUTH=строка_ключа_из_личного_кабинета_разработчиков_сбер
CERT_PATH=/path/to/certificate.pem_сертификат_для_работы

Нужно создать файл с разрешением .env и поместить туда исправленные под нужный проект данные конфигурации. Расположение поулчившегося файла должно быть в основной папке там, где README.md
